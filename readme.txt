Plugin for automated data migration from Kentico to WordPress. Convert Kentico to WordPress easily and fast.

== Description ==
One day you’ve found out that you would rather use WordPress than Kentico. In the result, an issue emerges - how to move all the content from Kentico based website to WordPress maximum quickly and safely? If you’re still thinking overcoming this problem, here’s the plugin to help.

= HOW THIS PLUGIN HELPS: =
- Free Demo Migration available (see the transfer result first by moving limited number of content to WordPress);
- Opportunity to choose where you want to move the data from Kentico to WordPress (Data “Mapping” option);
- Average migration time is from several minutes to several hours, depending on how many content items you’re up to move from Kentico to WordPress;
- The Complete data migration from Kentico to WordPress depends on the amount of content you want to move to your new WordPress website.


= WHAT WILL BE MOVED FROM KENTICO TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Kentico site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.


== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Kentico To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
